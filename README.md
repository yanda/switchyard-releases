# Switchyard Releases

Download and update feed for [Switchyard](https://github.com/yanda/chrome-router),
a macOS app that opens each link you click in the right Google Chrome profile.

- **Latest download**: grab the newest `Switchyard-x.y.z.zip` from this repo.
- **Updates**: installed copies update themselves via Sparkle using
  [`appcast.xml`](appcast.xml) served from this repo's GitHub Pages.

Install: unzip, drag `Switchyard.app` to `/Applications`, launch, and follow
the setup window.
